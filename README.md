This repo contains a collection of useful scripts and examples for working with the Engaging cluster.

See `slurm_scripts` for a selection of scripts for running interactive jobs with various amounts and types of compute resources.

See `vscode_setup` for an example of how to configure connecting your VSCode-based IDE to the cluster.
